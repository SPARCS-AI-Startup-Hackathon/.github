### etc
- spring security, jwt를 이용한 인증/인가 구현
- docker, docker-compose를 활용하여 컨테이너 및 이미지 관리
- spring webflux 를 이용해 스트리밍형 챗봇 구현
- 채팅관련 질문 및 답변을 핸들링 하기 위해 Redis 활용
- 직업을 찾고싶은 고령자, 직업추천, 자기소개서 작성 등을 위한 프롬프팅 처리
